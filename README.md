# Умный переводчик

Полный проект Android-приложения **Умный переводчик** — готов к загрузке на GitHub и автоматической сборке через GitHub Actions.

**Автор в README:** ne niger

## Что внутри
- Исходники приложения (Kotlin)
- UI: улучшенная визуальная версия (металлический градиент, Material-карточки)
- Поддержка выбора переводчика: Google ML Kit, LibreTranslate, Yandex Cloud (опционально), ChatGPT (опционально)
- Настройки: тема, выбор движка, ввод API-ключей (хранятся локально)
- GitHub Actions workflow для сборки APK (assembleRelease)

## Как использовать
1. Создайте новый репозиторий на GitHub (например `UmnyTranslator`).
2. Загрузите все файлы из этого архива в корень репозитория (Upload files).
3. Вставьте в Actions > workflow (автоматически запустится сборка). После успешной сборки скачайте APK в **Actions → Artifacts**.
4. При первом запуске приложения предоставьте разрешения: отображение поверх окон, доступность (Accessibility) и камера (если используете OCR).
